EnemyHP to Chat

Script [Not cheat], that will be send to team chat information, about enemy HP, after your death.

This script (This is not a cheat) uses the -netconport launch option, which allows you to connect to the game console and then read and send console commands. When you die, it writes in the team chat how much hp did you leave the enemy.


How to use:
Add to launch options this text:

                            
-netconport 10101

start the game and then, start a program, its all
